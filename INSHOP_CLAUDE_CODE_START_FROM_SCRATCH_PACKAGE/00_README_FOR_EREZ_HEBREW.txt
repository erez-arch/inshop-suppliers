חבילת התחלה מלאה לקלוד קוד - INSHOP
======================================

מה יש בזיפ:
1. SPEC.pdf - האיפיון המעודכן והמלא ב-PDF, בשם פשוט שקלוד קוד יבין.
2. SPEC.docx - אותו איפיון גם כוורד, לגיבוי/קריאה נוספת.
3. MOCK_DATA.xlsx - נתוני Mock למערכת הישנה דרך ExcelLegacyAdapter.
4. TEST_PACK/ - חשבוניות חיוב, חשבוניות זיכוי, אישורי תשלום, CSV, תמונות, PDF ו-JSON פלט AI צפוי.
5. 01_CLAUDE_CODE_FULL_PROMPT.txt - ההוראה המלאה להדבקה לקלוד קוד.
6. 02_FILE_MAP.txt - מפת קבצים קצרה.
7. CLAUDE_START_HERE.md - הוראה קצרה לקלוד לקרוא קודם.

איך להשתמש:
1. פתח תיקיית פרויקט חדשה וריקה.
2. העלה/העתק את הזיפ הזה לתיקייה.
3. חלץ את הזיפ בתוך התיקייה.
4. פתח Claude Code מתוך תיקיית הפרויקט.
5. הדבק לקלוד את כל הטקסט מתוך:
   01_CLAUDE_CODE_FULL_PROMPT.txt

חשוב:
- לא צריך להעלות לו עוד קבצים בנפרד אם חילצת את הזיפ.
- הקבצים נקראים בשמות פשוטים: SPEC.pdf, MOCK_DATA.xlsx, TEST_PACK.
- המטרה היא שהוא יבנה מערכת עובדת עצמאית מקצה לקצה, לא רק מסכים יפים.
- המערכת צריכה לעבוד בלי חיבור למערכת הישנה, דרך Mock/Excel Legacy Adapter.
